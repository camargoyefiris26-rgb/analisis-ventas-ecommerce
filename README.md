# Análisis de Ventas – Global Superstore (2011–2014)

Este proyecto presenta un análisis completo de datos de ventas utilizando **Python para el procesamiento de datos** y **Power BI para la visualización interactiva**, con el objetivo de extraer indicadores clave de desempeño (KPIs) y apoyar la toma de decisiones empresariales.

---

## Objetivo del proyecto

Analizar el comportamiento de las ventas del conjunto de datos **Global Superstore** para:

- Identificar tendencias de crecimiento
- Evaluar el rendimiento por categoría de producto
- Analizar los clientes más importantes
- Medir tiempos promedio de envío por segmento

---

---

## Procesos realizados

### Limpieza de datos
- Eliminación de valores nulos
- Conversión de tipos de datos
- Creación de columnas:
  - Año
  - Mes
  - Tiempo de envío en días

### Análisis Exploratorio
- Ventas por año
- Ganancias por categoría
- Top 10 clientes por ventas
- Tiempo promedio de envío por segmento
